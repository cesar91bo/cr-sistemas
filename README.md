# CR Sistemas — landing page

Landing estática hecha con HTML, CSS, JavaScript mínimo y Bootstrap 5 por CDN. Se puede abrir directamente con doble clic en `index.html`.

## Personalización rápida

- Reemplazá `+54 9 XXX XXX XXXX` y el enlace `https://wa.me/549XXXXXXXXXX` en `index.html` por el WhatsApp real (solo números, con código de país).
- Reemplazá `contacto@crsistemas.com.ar` por el email real.
- Guardá las capturas o imágenes en la carpeta `img` usando estos nombres:
  - `hero-dashboard.png`
  - `proyecto-repuestos-motos.png`, `proyecto-internet-cable.png`, `proyecto-delivery.png`, `proyecto-textil.png`
  - `experiencia-hospitales.png`, `experiencia-institutos.png`, `experiencia-crm.png`

Mientras no haya imágenes, la página muestra placeholders diseñados para cada bloque.

## Publicar gratis en GitHub Pages

1. Creá un repositorio en GitHub y subí estos archivos.
2. En el repositorio, abrí **Settings → Pages**.
3. En **Build and deployment**, elegí **Deploy from a branch**.
4. Elegí la rama `main`, la carpeta `/(root)` y guardá.
5. GitHub mostrará la URL pública de la página luego de unos minutos.
